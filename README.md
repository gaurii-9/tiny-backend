# Tiny Backend

A simple backend server built with Node.js and Express.

## Features

- JSON API endpoint: `/`
- JSON API endpoint: `/about`

## Technologies Used

- Node.js
- Express.js

## Installation

Clone the repository:

```bash
git clone https://github.com/gaurii-9/tiny-backend.git
```

Navigate to the project folder:

```bash
cd tiny-backend
```

Install dependencies:

```bash
npm install
```

## Run the Server

```bash
npm start
```

The server will start at:

```
http://localhost:3000
```

## API Endpoints

### GET /

Returns a welcome message.

Example response:

```json
{
  "message": "Hello, World!"
}
```

### GET /about

Returns information about the project.

Example response:

```json
{
  "name": "Gauri",
  "course": "Backend Basics"
}
```
